 > - показывает то, от какого процесса передаваь данные
Пример:

cat > 2.txt 
string
string
string
...
string


Также, с помощью этой штуки можно передавать инфу в другие файлы. Пример:

cat 2.txt > 3.txt 

Создастся тхт с теми же данными, что во 2.тхт

Если нужно добавить инфу не затирая старую, то можно следующее сделать:

cat 2.txt >> 3.txt


grep it < какой-то файл - вывод строк с указаным содержимым. В данном случае выведутся все строки, в
которых есть it(любое вхождение)
параметр -n дополнительно выводит нумерацию строк   

&& - аналог условия if, выполняется при положительном результате 

cat 2.txt && echo "THE TEXT WAS READ" - при чтении файла 2.тхт выведется сообщение 
								THE TEXT WAS READ


|| - аналог условия if, выполняется при отрицательном результате
