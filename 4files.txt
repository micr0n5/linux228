	Файл- некоторый набор блоков, который хранится на каком-нибудь носителе
Имя файла - конструкция, которая нужна для удобства пользователя


Каталог - является файлом, который содержит внутри себя другие файлы

Типы файлов:

Файлы устройств
Файлы каталогов
и т.д.


file - команда, которая применяется к файлам, показвыает тип файла

ls tty[1234] - [] в них пишется фильтр для поиска, ищет вхождение одно из символов 
в кв. скобках после введенных букв tty

Командой touch также можно менять метку времени в файле
Пример:
touch -t  199008161228 newfile.txt

так же можно метаданные одного файла перетащить в другой

touch -r 2.txt newfile.txt

mkdir [название] - команда для создания директори

mkdir -p [путь создания] - команда для создания цепочки директорий

rmdir [название] - удаление пустых каталогов

rm [название] - удаление файлов

rm -rf [название] - удаление каталогов

2 часть

копирование

cp [имя файла] [имя того, в который копируем] - команда для копирования


cp -u [имя файла] [имя того файла, в который копируем] - позволяет скопировать данные 
не затирая старые. Если файл, в который хотим скопировать будет новее(обновили его)
, то замены происходить не будет

При копировании директории копируется не её содержимое, а сама директория и её
содержимое

Перенос файлов из директорий 
cp -r hdd/* newdir   - пример

Изменение названий файлов

mv [старое название] [новое название]
