Начинаю своё обучение с навигации

whoami - узнать какой пользователь в системе "кто я?
pwd - узнать путь (место, где я сейчас нахожусь)
ls - узнать содержимое каталога

параметры к ls:

ls -l - выдаст дополнительно информацию о файлах, которые показал


cat - команда чтения (что-то прочитать, выводит текст файла в консоль)

cd - перемещение по каталогам

параметры к cd:

cd .. перенос на один уровень вверх

cd / перейти в корневой каталог



head -n  выводит n-е количество первых строк файла, где n - количество строк
tail -n выводит n-e количество последних строк файла, где n - количество строк

| позволяет выполнить сразу же две команды последовательно, например 

    ls -l | tail -5 выведет 5 последних файлов подробно

less - открыть файл с возможностью полистать его(для больших файлов актуально)
